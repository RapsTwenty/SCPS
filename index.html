How To Play

Android

  1. Download SCPS HOST.
  2. Buka aplikasi Virtual Host.
  3. Klik "RE-SELECT HOSTS FILE".
  4. Pilih file SCPS HOST.txt yang sudah didownload.
  5. Pastikan lingkaran menyala hijau.
  6. Login ke Growtopia!

Windows

  1. Tekan ⊞ Win+R dan buka C:\Windows\System32\drivers\etc
  2. Edit file hosts dengan Notepad.
  3. Tambahkan IP seperti di atas dan simpan.

iOS

  1. Download \"Surge 5\" dari App Store.
  2. Import konfigurasi dari file iOS di atas.
  3. Aktifkan VPN dan buka Growtopia.

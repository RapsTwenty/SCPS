How To Play

Android

  1. Download SCPS HOST.
  2. Buka aplikasi Virtual Host.
  3. Klik "RE-SELECT HOSTS FILE".
  4. Cari file SCPS HOST.txt yang sudah kamu download dan klik.
  5. Pastikan lingkaran menyala hijau.
  6. Sekarang kamu bisa login ke Growtopia!

Name Server: SCPS

Windows

  1. Tekan ⊞ Win+R untuk membuka jendela "Run".
  2. Ketik C:\Windows\System32\drivers\etc.
  3. Temukan file bernama hosts.
  4. Edit file "hosts" menggunakan Notepad.
  5. Tambahkan IP di atas.
  6. Simpan dan buka Growtopia.

Name Server: SCPS

IOS

  1. Unduh aplikasi "Surge 5" dari App Store.
  2. Tekan "OK".
  3. Klik "Default.conf".
  4. Klik "IMPORT" > Download Profile from URL.
  5. Masukkan URL dari tombol IOS host di atas lalu klik "OK" dan "Done".
  6. Tekan "SETUP", lalu izinkan VPN configuration.
  7. Buka Growtopia dan Connect!
